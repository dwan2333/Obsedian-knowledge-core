## 概述

AI戒习惯头箍是一个结合了**硬件工程、人工智能和神经技术**的闭环可穿戴系统。该设备能够主动监测用户环境，并在检测到不良习惯（抽烟、喝酒、吃垃圾食品）时进行**物理干预**。由于涉及佩戴在头部的物理外力和电流，**安全与电气隔离**是系统设计的重中之重。

## 系统架构概览

```
传感器层（头箍端）          →  通信层（Wi-Fi）  →  决策层（服务器端）
┌─────────────────┐        ┌──────────┐        ┌─────────────────┐
│ ESP32-CAM (视觉) │        │          │        │ FastAPI 数据摄取  │
│ 6轴IMU (运动)    │───────→│ ESP32    │───────→│ 视觉子代理       │
│ 麦克风 (声学)    │        │ Wi-Fi    │        │ 运动子代理       │
│ EMG (肌肉活动)   │        │          │        │ 管理器代理 →决定  │
└─────────────────┘        └──────────┘        └────────┬────────┘
                                                        │
干预层（头箍端）          ←  指令回传           ←────────┘
┌─────────────────┐
│ 1. 音频警告      │
│ 2. 头箍收紧      │
│ 3. 轻度电击      │
│ 4. 重度电击      │
└─────────────────┘
```

## 核心设计原则

- **边缘-服务器混合架构**：头箍端仅负责传感器数据采集和指令执行，繁重的AI推理在服务器端完成
- **多传感器融合**：不依赖单一传感器，通过视觉+运动+声学+肌电综合判断，最大程度减少误报
- **渐进式干预**：从口头警告逐步升级到电击，避免不必要的过度惩罚
- **安全优先**：每个反馈机制都有独立的安全保障（快速释放锁扣、光电隔离、低电流TENS）
