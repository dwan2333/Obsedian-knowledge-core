# 输出干预机制

*配套文档：[AI 戒习惯头箍 (Main)](<AI Habit Breaking Headband (Main).md>)*

---

## 概述

当系统判定不良习惯发生时，会触发物理和心理干预机制。该系统采用**渐进式升级协议** — 从最轻微的提醒逐步升级到最强烈的干预，避免不必要的过度惩罚。

---

## 音频提醒（第一级干预）

| 属性 | 详情 |
|---|---|
| **组件** | I2S微型扬声器 + DFPlayer Mini模块 |
| **功能** | 播放预录警告或AI生成的TTS语音 |
| **示例** | "放下香烟"、"不要喝这个" |
| **侵入性** | 最低 — 仅语音提醒 |

- 作为第一道防线，给用户一个自我纠正的机会
- 可以自定义警告语音内容

---

## 张力/拉伸机制（第二级干预）

| 属性 | 详情 |
|---|---|
| **组件** | 微型步进电机（NEMA 8）或微型伺服电机 |
| **传动系统** | 棘轮拨盘（类似BOA滑雪靴系统）+ 高拉力线 |
| **功能** | 电机卷起线材，物理收紧头箍 |
| **安全装置** | **必须配备物理机械快速释放锁扣** |

- 触发时，电机会卷起线材**在物理上收紧头箍**，产生不适感
- BOA拨盘系统的灵感来自滑雪靴 — 提供精确、可控的收紧力度
- **关键安全特性**：必须包含物理快速释放闩锁，允许用户在电机停转或过度收紧时**瞬间取下头箍**

---

## 电击反馈 / TENS（第三/四级干预）

| 属性 | 详情 |
|---|---|
| **组件** | 预制TENS（经皮神经电刺激）模块 + 光隔离继电器 |
| **功能** | 向皮肤提供剧烈但医学上安全的静电电击 |
| **强度级别** | 第三级：轻度电击 / 第四级：重度电击 |
| **安全原则** | **严禁从头构建高压电路** |

### 关键安全要求

1. **使用现成TENS电路** — 不要自行搭建高压电路，使用经过医疗认证的低电流TENS模块
2. **光电隔离器（Opto-isolators）是必须的** — 微控制器仅通过光耦来触发TENS电子开关，防止高压电涌回流烧毁微控制器
3. TENS提供的电击是**不舒服但安全的** — 与物理治疗中使用的TENS技术原理相同

---

## 升级协议总览

| 步骤 | 干预方式 | 强度 |
|---|---|---|
| **第1步** | 口头警告（音频） | 最低 |
| **第2步** | 头箍轻微收紧（张力） | 低 |
| **第3步** | 轻度电击（TENS低档） | 中 |
| **第4步** | 重度电击（TENS高档） | 高 |
