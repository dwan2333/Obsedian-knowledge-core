# 安全设计原则

*配套文档：[AI 戒习惯头箍 (Main)](<AI Habit Breaking Headband (Main).md>)*

---

## 概述

鉴于该设备佩戴在头部并涉及物理外力和电流，**安全与电气隔离是系统设计的重中之重**。每个反馈机制都有独立的安全保障措施，确保在任何故障情况下用户都不会受到伤害。

---

## 物理安全

### 快速释放锁扣

| 属性 | 详情 |
|---|---|
| **类型** | 物理机械式锁扣 |
| **位置** | 张力机制连接处 |
| **功能** | 允许用户在任何时刻**瞬间取下头箍** |
| **触发场景** | 电机停转、过度收紧、紧急情况 |

- 这是一个**纯机械装置**，不依赖电子系统 — 即使微控制器完全失效，用户仍然可以手动释放
- 设计灵感类似于滑雪绑定中的安全释放机制

---

## 电气隔离

### TENS模块安全规范

| 原则 | 详情 |
|---|---|
| **严禁自建高压电路** | 必须使用现成的、经过认证的低电流TENS电路 |
| **光电隔离器必须使用** | 隔离微控制器与TENS高压端 |
| **电流限制** | 使用低电流设计，符合医疗级TENS安全标准 |

### 光电隔离器（Opto-isolators）的作用

```
微控制器（低压端）    光电隔离器    TENS模块（高压端）
┌──────────┐       ┌──────┐       ┌──────────┐
│ GPIO信号  │──LED→│ 光耦 │→开关──│ TENS电路  │
│ 3.3V/5V  │       │      │       │ 高压脉冲  │
└──────────┘       └──────┘       └──────────┘
         ↑ 完全电气隔离 ↑
```

- 微控制器通过LED发光来"通知"TENS模块触发
- **两端没有任何电气连接** — 即使TENS模块故障产生高压尖峰，也不会影响微控制器
- 这是保护电子元件和用户安全的关键屏障

## 干预安全

### 多传感器融合减少误报

- 系统不依赖单一传感器做出干预决定
- 必须**多个传感器数据一致**才能确认不良习惯发生
- 校准模式建立用户的健康行为基线，进一步降低误报率

### 渐进升级原则

| 原则 | 说明 |
|---|---|
| **由轻到重** | 警告 → 张力 → 轻微电击 → 重度电击 |
| **给予机会** | 每一步之间给用户自我纠正的时间 |
| **避免过度惩罚** | 不在首次检测时就施加最强干预 |

## 安全设计检查清单

- [ ] 张力机制配备物理快速释放锁扣
- [ ] TENS使用现成的低电流医疗级模块
- [ ] 微控制器与TENS之间有光电隔离器
- [ ] 多传感器融合，不依赖单一传感器触发干预
- [ ] 升级协议从最低强度开始
- [ ] 校准模式可以记录和排除正常健康行为
