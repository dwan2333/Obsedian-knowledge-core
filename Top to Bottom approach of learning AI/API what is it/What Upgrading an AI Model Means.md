When companies release a new model, they are upgrading the underlying math and architecture in four main ways:

1. **Architecture (Parameters):** Adding more variables/weights so the model can grasp deeper logic and nuances.
    
2. **Context Window:** Expanding the model's memory so it can process hundreds of pages of text or code simultaneously.
    
3. **Inference Speed:** Optimizing the software (like quantization) and hardware routing so the model generates answers faster.
    
4. **Training Data:** Feeding the model cleaner, more professional, and highly complex data.
