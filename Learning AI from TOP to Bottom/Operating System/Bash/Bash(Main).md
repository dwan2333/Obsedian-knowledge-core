# Bash(Main)

### 🎓 The core concept
In Computer Science, we use the **Terminal** (or Command Line) to talk directly to the computer. 

*   **The Shell (Bash)**: This is the "face" or the "outer layer" (the shell). It's a program that interprets your text into something the computer can understand.
*   **The Kernel**: This is the "brain" or the "core" of the operating system. It manages your memory, files, and CPU.

### 🐧 Why "Bash"?
**Bash** stands for "Bourne-Again SHell" and is the standard for Linux and Mac systems. On Windows, we often use **PowerShell** or **CMD**, which do the same things but with slightly different "languages."

### 🛠️ Common Commands Reference

| Command (Bash) | Command (Windows) | What it does |
| :--- | :--- | :--- |
| `ls` | `dir` | List all files in the current folder |
| `cd <folder>` | `cd <folder>` | Change directory (move into a folder) |
| `mkdir <name>` | `mkdir <name>` | Make a new directory (folder) |
| `cat <file>` | `type <file>` | Show me the text inside a file |
| `rm <file>` | `del <file>` | Delete a file |

### 🤖 Why AI needs the Terminal
An AI agent uses these commands to "act" on your computer. It "types" into the terminal to run your Python code, read your notes, and manage your project.
